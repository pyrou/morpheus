# WHAT IS THIS ??

This is a little experiment I made to prove in live the danger of adding a composer package without checking its source code. Don't be afraid. This is just for demo purposes.

AND, this is completely un executed without an ENV variable named ALLOW_MORPHEUS_EXPERIMENT=1

The watcher is set in /tmp so it won't survice a reboot. But keep in mind that a malicious package could do anything,
and set a backdoor in any place, as a service, cronjob, etc.
